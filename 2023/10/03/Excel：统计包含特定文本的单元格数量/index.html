Excel：统计包含特定文本的单元格数量

Li Mingshuang
2023-10-03 (Updated: 2023-10-03)
解决方案

本文根据gpt3.5回答撰写

如果你希望统计包含目标文本的单元格数量，并计入合并单元格合并前的数量，可以使用以下的 VBA 代码来实现：

vba

Function CountMergedCellsContainingText(rng As Range, targetText As String) As Long
    Dim cell As Range
    Dim count As Long
    Dim mergedCell As Range
    Dim cellValue As String

    count = 0

    For Each cell In rng
        If cell.MergeCells Then
            ' 对于合并单元格，遍历合并区域内的每个单元格
            For Each mergedCell In cell.MergeArea
                cellValue = mergedCell.Value
                If InStr(1, cellValue, targetText, vbTextCompare) > 0 Then
                    ' 如果合并单元格内包含目标文本，增加计数
                    count = count + 1
                    Exit For ' 终止循环，只计数一次
                End If
            Next mergedCell
        Else
            ' 对于非合并单元格，检查单元格的值是否包含目标文本
            cellValue = cell.Value
            If InStr(1, cellValue, targetText, vbTextCompare) > 0 Then
                count = count + 1
            End If
        End If
    Next cell

    CountMergedCellsContainingText = count
End Function

这个修正后的 VBA 代码会遍历合并单元格中的每个单元格，检查它们是否包含目标文本，并在合并前计数满足条件的单元格。如果合并单元格中的任何单元格包含目标文本，将增加计数并退出合并单元格的循环以避免重复计数。

你可以在 Excel 工作表中使用 CountMergedCellsContainingText 函数来执行这个操作，如下所示：

excel

=CountMergedCellsContainingText(A1:B2, "目标文本")

这将统计 A1 到 B2 范围内包含 “目标文本” 的合并单元格数量，同时考虑合并前的数量。

为什么“对于合并单元格，遍历合并区域内的每个单元格”

ChatGPT
对于合并单元格，需要遍历合并区域内的每个单元格，主要是因为合并单元格实际上是一个单一的视觉单元，但在 Excel 内部，它们是由多个单元格组成的。因此，如果你想要在合并单元格中查找特定的文本或执行其他操作，你需要遍历合并区域内的每个单元格来检查其内容。

合并单元格的特点包括：

  1. 外观上的统一性： 合并单元格在外观上看起来像一个单元格，通常用于创建表格中的标题、标签或其他内容，以提高可视化效果。

  2. 内部的多个单元格： 从数据存储的角度看，合并单元格实际上是由多个单元格组成的。只有合并单元格中的第一个单元格包含真正的数据，其他单元格可能为空。

因此，如果你要在合并单元格中执行操作，例如查找特定文本，你需要遍历合并区域内的每个单元格，以确保不会漏掉包含目标文本的单元格。这可以通过循环遍历合并单元格内的每个单元格来实现，然后检查它们的内容以确定是否满足条件。

在上面提供的修正后的 VBA 代码中，我们遍历合并单元格的每个单元格，以确保不会错过任何包含目标文本的情况，并在找到满足条件的单元格后退出循环，以避免重复计数。这样可以确保在合并单元格中准确地计算包含目标文本的单元格数量。
